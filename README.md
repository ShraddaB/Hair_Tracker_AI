# Hair Tracker AI

## Overview
Hair Tracker AI is a Python-based project that analyzes hair health and hair type using machine learning models. It can classify hair type, detect hair-related diseases, and predict hair health.

## Features
- Hair type classification
- Hair disease detection
- Model evaluation and training scripts
- Easy-to-use Streamlit app for user interaction
